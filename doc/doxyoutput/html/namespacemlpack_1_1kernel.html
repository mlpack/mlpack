Detailed Description

Kernel functions.

This namespace contains kernel functions, which evaluate some kernel function for some arbitrary vectors and of the same dimension. The single restriction on the function is that it must satisfy Mercer's condition:

for all square integrable functions .

The kernels in this namespace all implement the KernelType policy. For more information, see The KernelType policy documentation.
